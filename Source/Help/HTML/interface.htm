The User Interface

BoobyTrap's user interface has three main areas:

The Play Area

The large area on the left main window is where the game grid is displayed. See How to play the game for details of how you interact with game grid.

The Button Bar

The button bar appears down the right hand side of the main window. It contains the following command buttons:

  • New Game
    This button starts a new game – if there is a game in progress it abandons that game.
  • Surrender
    This button is only available when a game is in progress (i.e. when the timer is running). You click it to abandon the current game and get a display of where all the booby traps were.
  • High Scores
    This button displays the High Scores dialogue box for the current play area.
  • Skill Level
    The Skill Level button displays the Skill Levels dialogue box that allows you to change the grid size or difficulty level and define new playing grids.
  • Help
    This button displays this help file.
  • About
    The About button displays the program's author and copyright information.
  • Exit
    The Exit button quits the program.

The Status Bar

The status bar appears across the bottom of the main window. It is divided into four main panels:

  1. The status panel gives details of the status of the game – it tells you how to start a new game, if the game is in progress and if you've won or lost.
  2. The timer panel displays elapsed time when a game is in progress or the time taken for the game once the game is over.
  3. The flags panel shows how many flags are available for you to place – this reduces as you place the flags.
  4. The skill level panel displays the current game's grid size and number of booby traps hidden under the grid.
