Skill Levels Dialogue Box

This dialogue box allows you to:

  • select a new difficulty level for the current game grid
  • edit the custom difficulty level for a game grid
  • select a different game grid
  • define a new game grid size
  • delete an existing game grid size

These options are explained below.

Adding a new game grid size

To add a new game grid size click the Add new play area button and enter the required number of columns and rows in the resulting New Play Area dialogue box. The new game grid now appears in the Playing area list box.

Deleting a game grid size

To delete a game grid size, select the grid size you want to delete from the Playing area list box and click the Delete play area button. Confirm that you want to go ahead with the deletion in the resulting confirmation dialogue box. Note that you can't delete the game grid currently in use and you can't delete all the game grids.

Selecting a game grid size to use

To select a game grid size to use for your next game, simply select the required grid size from the Playing area list box.

Selecting a difficulty level

To choose a different difficulty level for you next game choose the required level of difficulty from the Difficulty drop-down list. The number of booby traps in the game are displayed alongside the drop-down list. If you select the Custom... difficulty level you will be invited to edit the number of booby traps. If you don't want to do this just cancel the resulting dialogue box.

Editing the custom difficulty level

Choose the Custom... item from the Difficulty drop-down list. This will display the Edit Number of Booby Traps dialogue box where you should enter the number of booby traps you require. On accepting this input the number of booby traps alongside the drop-down list will be updated to the new value.
