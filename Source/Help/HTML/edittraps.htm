Edit Number of Booby Traps Dialogue Box

This dialogue box appears when you select the Custom difficulty level in the Skill Levels dialogue box.

To retain the offered number of booby traps you can simply press Cancel.

To alter the number of booby traps you should enter a number in the range displayed in this dialogue box and click OK. If the number entered is within the required range you will be returned to the Skill Levels dialogue box and the number of booby traps you entered will be used for the Custom difficulty level. If the value is not within the required range you will be informed and this dialogue box will not close.
