New Play Area Dialogue Box

This dialogue box appears when you click the Add new play area button in the Skill Levels dialogue box.

It allows you to enter the number of columns and rows to appear in the new game grid. You are limited to between 6 and 32 columns and 6 to 18 rows.

Once the values have been entered click the OK button to accept your input. If you have entered valid values you will be returned to the Skill Levels dialogue box. If any value are in error you will be told and this dialogue box will not close.

Clicking Cancel aborts the entry and no new game grid will be defined.
