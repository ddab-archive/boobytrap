High Scores Dialogue Box

This dialogue box displays the leader board for the current game grid.

The size of the game grid is displayed in the dialogue's title bar while the names and times for the three best scores for each of the grid's difficulty levels (Easy, Medium, Hard and Custom) are displayed in the body of the dialogue box. The Close button closes the dialogue box.

All the leader board entries can be cleared by clicking the Restore button and confirming the action in the resulting confirmation dialogue box.
